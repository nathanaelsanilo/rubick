# Rubick

A Node.js project for transform object shape to another object shape.

## Background Story

I wanted to have a mapper library similar to ModelMapper in the Java ecosystem. Then, I remembered the class-transformer library which does the same thing in the Node.js environment. I thought it would be a great project to build my own library.

Since this is new for me, I decided to use Gemini as a "coding buddy" to discuss ideas with. My goal in creating this is to gain new experiences outside of my 9-5 job. Through this project, I am diving deep into the prototype chain and relearning JavaScript fundamentals.

## Installation

```bash
pnpm install
```

Build

```bash
pnpm build
```
